# **О проекте**
- Проект представляет собой конструктор, который позволяет создавать свои файлы автоматизации, которые в свою очередь выполняют разные действия
> [!IMPORTANT]
> ## Что нужно для начала работы с программой?
> - Достаточно лишь [скачать](https://www.python.org/downloads/release/python-3137/) с официального сайта Python, и установить флажок на "Add Python 3.17.3 to PATH"
> - Всё! С программой можно работать
# Ответы на вопросы:
- ## Будут ли выходить обновления на конструктор?
> Сложно сказать. Пока не планирую чего-то грандиозного. Если мне прийдёт очередная идея в голову про команды, то я обязательно её добавлю в конструктор
- ## На каких ОС поддерживается конструктор?
> Программа поддерживает пока только Windows
- ## Если у меня установлен Python, то его нужно переустанавливать?
> Смотря по случаю. Если у вас выдаёт ошибку, то переустановить стоит по ссылке выше
- ## Можно ли мне предложить идеи для команд?
